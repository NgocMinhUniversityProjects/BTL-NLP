## Dataset used

### Traditional sentiment analysis

This task focuses on "tradtional" sentiments like happy, sad, angry, etc.

The two datasets our group agrees on for this task is

1. [TweetEval](https://www.kaggle.com/datasets/thedevastator/tweeteval-a-multi-task-classification-benchmark)
2. [Twitter Sentiment Analysis](https://www.kaggle.com/datasets/jp797498e/twitter-entity-sentiment-analysis)

### Depressive text analysis

This task focuses on "expanded" or critical and vulnerable emotions like Suicidal, Depression, etc

The two datasets our group agrees on for this task is

1. [Classified Twitter posts](https://www.kaggle.com/datasets/munkialbright/classified-tweets)
2. [Derived dataset from Murarka et al., Nikhileswar Komati and Suchintika Sarkar](https://www.kaggle.com/datasets/priyangshumukherjee/mental-health-text-classification-dataset)
